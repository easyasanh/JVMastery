# Fund and Games

Fund and Games is a static investment idle clicker built for GitHub Pages. Start as a retail trader, research trades for capital, buy upgrades, unlock deeper fund mandates, and eventually close the fund for permanent Track Record bonuses.

## Features

- Clicker loop with passive income upgrades
- Ten progression layers from Retail Account to Capital Engine
- Upgrade categories for research, people, and strategy
- Upgrade ownership milestones at 10, 25, 50, 100, and 250
- Buy modes for 1x, next milestone, or maximum affordable purchases
- Local save persistence with `localStorage`
- Offline progress for up to 8 hours
- Prestige loop via Close the Fund
- No build step required

## Local usage

Open [`index.html`](/Users/Andrew/Documents/New%20project/index.html) directly in a browser, or serve the folder with any static server.

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy to GitHub Pages

1. Push this repository to GitHub.
2. In the repository settings, enable GitHub Pages.
3. Set the source to the root of the `main` branch.
